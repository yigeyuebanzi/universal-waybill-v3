# DeepSeek 使用说明

## 接入点

V3 在两个位置提供 AI 建议：

- 异常上报：根据异常描述建议异常类型和严重度。
- 审批详情：根据历史审批记录给审批意见建议。

## 环境变量

```env
DEEPSEEK_BASE_URL
DEEPSEEK_API_KEY
DEEPSEEK_MODEL
```

## 原则

- AI 输出必须标注“AI 建议，需人工确认”。
- AI 不自动创建工单。
- AI 不自动审批。
- AI 不自动执行库存或赔付动作。
- AI 调用失败或超时不阻塞主流程。

## 可解释性

审批建议会附带最近历史审批记录作为依据，避免黑箱结论。
